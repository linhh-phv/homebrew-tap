# homebrew-tap

Homebrew tap for [linhh-phv](https://github.com/linhh-phv)'s CLI tools.

## Install

```bash
brew tap linhh-phv/tap
brew install <tool>
```

Or install directly without tapping:

```bash
brew install linhh-phv/tap/<tool>
```

## Available formulae

| Tool | Description |
| --- | --- |
| [`lps`](https://github.com/linhh-phv/lps) | List & manage processes by listening port, grouped by project |

## Update

```bash
brew update
brew upgrade <tool>
```

## Uninstall

```bash
brew uninstall <tool>
brew untap linhh-phv/tap
```
